# Licensing decision checklist

Do not publish a release without choosing licenses deliberately.

Questions to settle:

- Code license for the Streamlit app and simulation scripts.
- License for manuscript source.
- License for generated figures.
- License for educational text and exercise material.
- Whether third-party screenshots or logos are included.

Common patterns to evaluate with appropriate advice:

- permissive software license for code,
- Creative Commons license for text and figures,
- separate notices for third-party material.

This starter does not choose a license on the author's behalf.
